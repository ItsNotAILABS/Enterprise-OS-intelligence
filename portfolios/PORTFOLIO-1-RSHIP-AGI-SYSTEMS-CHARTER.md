## Mathematical Foundations

### Core Algorithms in Aviation AGIs

#### M/D/1 Queue (AEROLEX — Gate Throughput)
```
ρ = λ/μ                           (utilization ratio)
Lq = ρ² / (2(1 − ρ))              (queue length)
Wq = ρ / (2μ(1 − ρ))              (wait time)
τ_delay = Lq × service_time_avg   (expected delay)
```

#### Samn-Perelli Fatigue Model (CREWEX)
```
f(w, h) = 1 + 0.15w + τ(h) / restQuality
where:
  w = waking hours elapsed
  τ(h) = circadian trough factor (1.5 if 2≤h≤6, else 0)
  restQuality = [0,1] pre-duty rest quality score
Scale: 1 (fully alert) → 7 (completely exhausted)
Ground threshold: f ≥ 5.5
```

#### Kelly Criterion Fare Allocation (VISITEX)
```
f* = (bp − q) / b
where:
  b = revenue odds (fare / cost basis)
  p = booking probability (learned from historical acceptance)
  q = 1 − p
  max(f*, 0.25)   (cap at 25% allocation)
```

#### Gravity Model Tourism Demand (VISITEX)
```
D(i,j) = G × (P_i × P_j) / d²_ij × S(t)
where:
  G = calibrated gravitational constant (1×10⁻⁸)
  P_i, P_j = population of origin/destination
  d_ij = distance in km
  S(t) = seasonality: 1 + 0.3 × sin(2π × month/12)
```

#### φ-Weighted Passenger Priority (PASSEX)
```
priority(p) = tier_weight × φ^(-frustration_quantile)
where:
  tier_weight: VIP=φ², preferred=φ, standard=1
  frustration_quantile: [0,1] normalized frustration score
```
